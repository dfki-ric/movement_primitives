movement_primitives.spring_damper.SpringDamper#

class movement_primitives.spring_damper.SpringDamper(n_dims, dt=0.01, k=1.0, c=None, int_dt=0.001)#

Bases: PointToPointMovement

Spring-damper system.

This is similar to a DMP without the forcing term.

Parameters:
n_dimsint

State space dimensions.

dtfloat, optional (default: 0.01)

Time difference between DMP steps.

kfloat, optional (default: 1)

Spring constant.

cfloat, optional (default: 2 * sqrt(k) (critical damping))

Damping coefficient.

int_dtfloat, optional (default: 0.001)

Time difference for Euler integration.
